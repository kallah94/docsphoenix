    Phoenix v1.4.15 mix local.phx View Source

    Updates the Phoenix project generator locally.

    mix local.phx

    Accepts the same command line options as archive.install hex phx_new.

    Note: Older versions of this task (up to and including 1.4.8) do not fetch the latest version from hex. If your phx_new archive is older than 1.4.9, it's necessary to call mix archive.install hex phx_new manually at least once.
